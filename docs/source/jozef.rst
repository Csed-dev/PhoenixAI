Jozef Module Documentation
==========================

This section documents the contents and purpose of the `jozef` directory.

Contents
--------

.. toctree::
   :maxdepth: 1

- **generated_files**: Contains files generated automatically by the project.
- **training_data_preparation**: Includes scripts for preparing training data, such as:
   - `convert_training_data.py`
   - `scrape_pylint_docs.py`
   - etc.

Each script in the `jozef` folder serves a specific function in data handling, model training, or analysis for the Phoenix AI project.
